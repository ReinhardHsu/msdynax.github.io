Dynamics AX 2012 R2 AOT Overview

撰写于 2015-01-05 修改于 2018-05-01 分类 Microsoft Dynamics AX 标签 Microsoft Dynamics AX

Microsoft Dynamics AX中，应用程序对象树（AOT）包含所有用于构建Microsoft Dynamics AX的元素，如类，表，窗体，等等的定义。本主题提供了AOT的概述，并定义了顶级节点。

要创建在AOT中的新元素，用鼠标右键单击相关的节点，然后单击新建 。此外，拖动和拖放操作可用于多种元素。

在顶级节点的所有元素都有：

  • 一个快捷菜单。打开快捷菜单，右键单击一个元素。欲了解更多信息，请参阅快捷菜单命令：AOT 。
  • 属性。要查看某个元素的属性和属性值，请右键单击该元素，然后单击属性 。显示属性表。欲了解更多信息，请参阅应用程序对象的属性 。

在AOT包含下表中描述的顶层节点。









































































































节点描述
Data Dictionary包含那些组成数据库的数据类型和表。还包控制数据访问的对象。它包含以下子节点：



表 ：包含Microsoft Dynamics AX的数据表。

映射 ：使您可以在密切相关（但不相同）的表字段和方法之间创建关联。

视图 ：使您可以从不同的表联接的数据，然后选择要显示哪些字段。

扩展数据类型 ：那些扩展自基本数据类型或另一个扩展数据类型的数据类型。

基地枚举 ：枚举类型包含一个文字的列表。

许可证代码 ：确定要提供给公司的Microsoft Dynamics AX的功能组件。

配置键 ：允许管理员为所有用户启用或禁用应用程序中的功能。

安全密钥 ：安全密钥在Microsoft Dynamics AX 2012中已经过时，只存在一个代码升级过程中使用以供参考。基于角色的安全，是一个新的安全架构。有关新的安全框架的更多信息，请参阅最新消息：开发者安全性在Microsoft Dynamics AX 2012 ，并在AOT为开发者基于角色的安全性 。

表集合 ：一个数据表的集合，它包含那些总是在企业之间共享的数据。

观点 ：我们用来为报表模型组织信息的表的集合。
Macros包含用于标准应用的宏的源代码。除了查看现有的代码，你可以添加你自己的宏。
Classes包含应用程序（X + +）类的源代码。

您还可以使用系统类（也称为核心类）。它们被列在系统文档\类节点。

Forms用户界面的对话框，用于访问数据库。
Parts配件包含您可以用于检索和显示数据集合的控件。欲了解更多信息，请参阅部件 。
Data Sets提供了一个通用的数据访问层，它允许外部表示层绑定到Microsoft Dynamics AX的表和数据类型。欲了解更多信息，请参见数据集的企业门户 。
SSRS Reports包含Microsoft Dynamics AX中的SQL Server Reporting Services报表。
Reports使用户能够从数据库中打印或显示摘要信息。
Visual Studio Projects包含在Visual Studio中创建和使用应用程序浏览器添加到Microsoft Dynamics AX中的项目。可以添加到这个节点的项目类型包括Dynamics AX的示范项目，使用C Sharp项目，Visual Basic项目，Web应用程序项目，和Analysis Services项目。欲了解更多信息，请参见Visual Studio集成和如何添加一个Visual Studio项目的AOT 。
Report Libraries用于存储的Microsoft Dynamics AX 2009的正在升级为Microsoft Dynamics AX的2012 AOT环境中的SQL Server Reporting Services报表库。
Queries作为对窗体和报表的记录源。
Jobs通常认为，是用来测试新代码的小X + +程序。
Menus包含您希望最终用户看到的菜单。
Menu Items包含了可以在菜单中显示的菜单项的完整清单。菜单项充当一个更高的抽象层为窗体，报表，等等。
Web包含关于Web开发的对象。
Services包含由Microsoft Dynamics AX的公开的服务。
Service Groups包含经常食用和共同管理的服务的集合。服务组中的所有服务发表在一个单一的WSDL文件。
Workflow包含用于创建工作流配置的工作流模型元素。此节点包含类别 ， 任务 ， 认证 ，和模板 。欲了解更多信息，请参见实施工作流程Microsoft Dynamics AX的 。
Security包含用于实现应用程序安全性的对象，如角色和权限。
Resources包含引用的图像和动画文件。
Label Files 包含为所有用户界面元素存储标签的标签文件。欲了解更多信息，请参见标签编辑器 。
References包含对微软.NET程序集和到外部Web服务的引用。这两种类型的引用可以在X + +语句中使用。
Help Documentation Sets 指定帮助服务器上​​的文档集。
System Documentation包含表示系统（内核）类，函数，表，等项目。
