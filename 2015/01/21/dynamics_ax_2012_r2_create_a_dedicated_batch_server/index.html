Dynamics AX 2012 R2 Create A Dedicated Batch Server

撰写于 2015-01-21 修改于 2018-05-01 分类 Microsoft Dynamics AX 标签 Microsoft Dynamics AX

安装额外AOS的另一原因，是要创建一个专用的Batch服务器。

AOS实例在处理batch job时，会影响它的响应速度。安装一个专用Batch服务器，可以解决这个问题。

批处理服务器不能加到集群中，用户也不应连接到它。

  1. 进入System Administrator>Setup>System>Server Configuration。
  2. 在左边的列表，选中要设为Batch服务器的AOS。
  3. 在右边的明细中，选中Is Batch Server。
  4. 在Batch Server Groups中，选择Batch Group。
  5. 取消其他AOS的Is Batch Server。

当一切都完成后，打开System Administrator>Common>Users>Online Users。你会看到Server Name 为MSDynAX_ApplicationServer的服务器上，有许多工作线程，这就是正在执行的批处理。
